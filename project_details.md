SpanStay: Project Portfolio & Freelance Pitch Guide
This document breaks down the SpanStay project to help you showcase it effectively in your portfolio, on your resume, or when pitching to potential freelance clients.

🚀 Project Overview
Project Name: SpanStay
Type: Full-Stack Accommodation Booking Platform

Project Description (Elevator Pitch):
SpanStay is a modern, high-performance, full-stack hotel and accommodation booking application built with Next.js 16. It delivers a seamless user experience for discovering properties, interactive map-based searching, and secure end-to-end booking management. The platform features robust user authentication, complex state management, and an aesthetically pleasing, responsive UI enhanced with micro-animations.

🛠️ Technical Skills & Tech Stack
When showcasing this project, highlight these specific technologies as they demonstrate your expertise in modern web development standards.

Frontend & UI/UX
Core Frameworks: Next.js 16 (App Router), React 19
Styling: Tailwind CSS v4 (Modern utility-first CSS)
State Management: Redux Toolkit (@reduxjs/toolkit, react-redux)
Animations: Framer Motion (Fluid page transitions and micro-interactions)
UI Components:
react-map-gl: Interactive mapping capabilities
swiper & yet-another-react-lightbox: Advanced image carousels and galleries
react-day-picker: Custom date selection for bookings
lucide-react: Consistent, scalable iconography
Data Visualization: Recharts (For analytics/dashboards)
Backend & Database
API & Routing: Next.js Serverless API Routes
Database: MongoDB
ORM / ODM: Mongoose (Structured data models for Hotel and Booking)
Authentication: Clerk (@clerk/nextjs for enterprise-grade secure auth)
Form Handling & Utilities
Form Management: React Hook Form
Schema Validation: Zod
API Fetching: Axios
Date Manipulation: Date-fns
Notifications: React Hot Toast
✨ Key Features to Highlight
When writing case studies or speaking with clients, focus on these core features you built:

Secure Authentication Flow: Integrated Clerk for seamless user sign-up, login, and session management.
Interactive Map-Based Search: Implemented a dynamic map feature allowing users to visually explore hotel locations and pricing in real-time.
Advanced Filtering & Search: Built complex search functionality allowing users to narrow down accommodations by price, location, rating, and amenities.
End-to-End Booking System: Developed a comprehensive booking flow capturing check-in/out dates, guest counts, and total price calculation, tracking statuses from "pending" to "confirmed".
Modern, Responsive Design: Crafted a mobile-first, highly responsive user interface with Tailwind CSS, enriched with smooth animations via Framer Motion.
Robust Data Validation: Utilized Zod and React Hook Form to ensure high-quality user inputs and prevent client-side errors.
